# シフト表メーカー

ブラウザだけで動く、無料のシフト表作成ツール。インストール不要・ログイン不要。
部署ごとの管理、記号／時間表示の切り替え、曜日パターンの自動反映、祝日・会社休業日の色分け、印刷／PDF、CSV書き出しに対応。

- 作者: 茶々丸（業務自動化）
- note: https://note.com/cha_cha_maru_ai

## 使い方
`index.html` をブラウザで開くだけ。入力内容はお使いのブラウザ内に自動保存されます。

## 公開URL
GitHub Pages で公開しています → （公開後にURLをここに記載）

## データについて
データはブラウザのローカル（localStorage）に保存されます。履歴・サイトデータを消すと入力も消えます。別端末とは共有されません。
